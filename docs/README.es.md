# 🎨 WinPaint Linux

Un clon fiel de **Microsoft Paint** para Linux.

## Características

- **Herramientas clásicas**: Lápiz, pincel, línea, rectángulo, elipse, borrador, relleno, texto
- **Selección**: Rectangular y libre con marching ants
- **Colores**: Paleta clásica de 28 colores + selector personalizado
- **Historial**: Deshacer/Rehacer ilimitado
- **Zoom**: 12.5% hasta 800%
- **Formatos**: PNG, JPEG, BMP
- **Atajos de teclado**: Idénticos a Paint (Ctrl+N, Ctrl+Z, etc.)

## Instalación rápida

```bash
git clone https://github.com/tu-usuario/winpaint-linux.git
cd winpaint-linux
make install-deps-debian  # o -fedora
make install
make run
```
